To compile, run the command:
	javac Main.java

alternatively, you may compile each program individually by running the commands:
	javac RSAGenKey.java
	javac RSAEncrypt.java
	javac RSADecrypt.java

To run all the programs together run the command:
	java Main

alternatively, you may run each program individually by running the commands:
	java RSAGenKey
	java RSAEncrypt
	java RSADecrypt
